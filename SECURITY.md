# Security Policy

If you believe you have found a security vulnerability in this project, please do not open a public issue.

Instead, report it privately to:

`dev@unipiazza.it`

Please include, when possible:

- a short description of the issue
- the affected area or file
- reproduction steps
- possible impact
- any suggested mitigation

We will review the report and respond as soon as reasonably possible.

## Supported Versions

At the moment, security fixes are handled on the latest version of this repository.
